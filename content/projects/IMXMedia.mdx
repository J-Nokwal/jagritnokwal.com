**IMX Media** is a **freelance** project: a fast, modern website that presents the brand and its creative services with a clean visual language. The build focuses on performance, responsiveness, and motion to keep the experience engaging across devices.

**Key Features:**
link: https://www.imxmedia.in/

- 📱 Fully **responsive** layout with mobile-first breakpoints  
- 🎞️ **Complex animations** (section reveals, hover micro-interactions, smooth transitions)  
- ⚡ Optimized assets with **Next.js Image** and route-level code splitting  
- 🔎 **SEO-ready** meta, Open Graph tags, and structured content  
- ♿ Semantic HTML and accessible components for better Lighthouse scores  
- 🚀 **Vercel** deployment with automatic preview builds and caching
